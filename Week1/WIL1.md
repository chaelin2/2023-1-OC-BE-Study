# 기초 백엔드 스터디

## MVC와 템플릿 엔진
   Model : 데이터와 비즈니스 로직 관리
     View : 레이아웃, 화면 처리
     Controller : 명령을 모델과 뷰 부분으로 라우티

## API와 서버
     json이라는 데이터 구조 포맷으로 클라이언트에 데이터 전달

## RESTFful
    REST는 Representational State Transfer의 약자.
      자원을 URI로 표시하고 해당 자원의 상태를 주고 받는 것
    RESTful - REST를 구현하는 웹 서비스를 나타태기 위해 사용되는 용어
